Let's create our First Program

  • Open Eclipse IDE, click on File icon>New>Other. Search Java Project and create one project.
  • After project creation, create one package inside src folder, here I create package named com.learn.
  • Inside package create class named HelloWorld and select public static void main checkbox below in the screen and create.
  • You can see now file is created with some boilerplate code.
                package com.learn;

                public class HelloWorld{{ob}}
                
                public static void main(String[] args){{ob}}
                    System.out.println("Hello World");
                    {{cb}}
                {{cb}}

HelloWorld.java


  • Run the program you will get Hello World as output in console.
