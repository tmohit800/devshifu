Follow below steps to start working on angular on your local system:

  • Angular requires one active Node.js version. To download Noad.js, visit nodejs.org
  • After download, install nodejs. Once installation completes, go to cmd(Command Prompt) and run node -v command. If version appeared on the cmd it shows that our node is successfully installed.
  • While installing Node automatically sets the environment variable, but if node -v command throws error, go to environment variable create new variable inside path and set the node js path which will be "C:\Program Files\nodejs".
  • Now close the cmd and open again and again run node -v command. The installed version will appear.
  • Now we will install Angular CLI, a command line interface for the angular. To install this run npm install -g @angular/cli, here -g refers to global.
  • After installing Angular CLI, go to cmd and run command ng version. If angular CLI is successfully installed, its version will appear on screen.

Angular is installed successfully now let's learn some basic commands.

  • ng new my-app: this will create one new angular project with name 'my-app'.
  • ng g c componentName: this will generate new component, g-> generate, c-> component
  • npm i: to install packages from package.json, i-> install
  • ng g s serviceName: this will generate new service, g-> generate, s-> service
  • ng serve: runs the project on localhost, by default on 4200 port
  • ng serve --port=4300: runs the project in given port, 4300 in this case
  • ng serve --host 0.0.0.0: runs the project as a host server, you can access your project through any other device, by using url in given format-> ip of your system:port(4200 by default)
