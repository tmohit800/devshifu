First we will see how to work with Bootstrap

  • Install bootstrap by using command npm i bootstrap --save
  • Go to angular.json file inside styles array (of build attribute not of test), paste these path "src/styles.css", "./node_modules/bootstrap/dist/css/bootstrap.min.css"
  • Bootstrap will start to work, if you want to use bootstrap.min.js also, you need to install jquery(till Bootstrap 4, Bootstrap 5 is removing support for jquery).
  • To install jquery, run command npm i jquery --save
  • Again go to angular.json file inside scripts array (of build attribute not of test), paste these path "./node_modules/jquery/dist/jquery.min.js", "./node_modules/bootstrap/dist/js/bootstrap.min.js". Don't change the sequence of import as jquery need to be imported before because bootstrap require jquery.
  • In case, this doesn't work, cross check whether the path is correct or not and required paths are inside 'build' object of angular.json not of 'test' object.

Now we will see how to start with material

  • Install @angular/material using command npm i @angular/material
  • Go to styles.css file and paste the import @import "~@angular/material/prebuilt-themes/indigo-pink.css";. Material requires one theme to start, you can use theme according to your choice.
  • Unlike Bootstrap, to use any angular material component you need to import it in module.ts of component. Till Angular 9 you can import module like this import {{ob}} MatButtonModule {{cb}} from '@angular/material'; but from angular 9 you have to import like this import {{ob}} MatButtonModule {{cb}} from '@angular/material/button';. After importing put the Module into imports array of module.

Jquery is rarely used when we work in angular, still to use it in your component you can follow below steps.

  • Install Jquery and put it in scripts we have already done this part in bootstrap installation in above section.
  • Go to html of component and create a button with class 'cls'. <button class="btn" type="button">Click Me!</button>
  • Go to component.ts and import jquery like this import * as $ from 'jquery'
  • Go to component.ts file and inside ngOnInit function paster this code. $('.btn').click(function() {{ob}} alert("Button clicked"); {{cb}})
  • Click on button, alert with content Button clicked will come.

Getting error in JSON path? To use json in your module, you have to change one property.

  • In your tsconfig.json inside compileOptions add one property "resolveJsonModule": true
  • If project is running stop it and start again. Error in JSON path will be resolved.
