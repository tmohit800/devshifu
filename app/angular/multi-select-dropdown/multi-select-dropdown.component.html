  • Add "NgMultiSelectDropDownModule.forRoot()" in imports section of your parent module.ts file. Now let's understand some part of the code:

  • So we have installed the package so that we can use it after that we import it on parent module so every child component is aware of it and can use it.
  • dropdownList will populate the starting values in the dropdown.
  • selectedItems will select initial values.
  • dropdownSettings will configure your dropdown which implements an interface named IDropdownSettings.
    • single selection: decides whether to select multiple values from dropdown or not.
    • idField: dropdown will select values on behalf of this id, the value of selection is binded with id. If two entry of dropdown have same id then they will be checked and unchecked at same time.
    • textField: this decides what to show in front of the checkboxes.
    • selectAllText & unSelectAllText: text on selectAll and unselectAll link.
    • itemsShowLimit: decides how many selected items need to show on the dropdown. Set it 0 and you will get rid of the selected value showing over the dropdown.
    • allowSearchFilter: decides whether to give search functionality or not.
    • Want to remove select all feature, just add 'enableCheckAll' attribute and set it to false.
